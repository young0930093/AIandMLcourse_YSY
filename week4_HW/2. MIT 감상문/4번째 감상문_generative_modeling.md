# MIT 6.S191 - Deep Generative Modeling 감상문

이번 **MIT 6.S191 - Deep Generative Modeling** 발표 자료를 보면서 가장
먼저 인상 깊었던 점은, 생성 모델링이 단순히 "없는 것을 만들어내는
기술"이 아니라 **데이터의 본질적인 구조를 이해하려는 시도**라는
점이었다. 처음 슬라이드에서 실제 사람처럼 보이지만 존재하지 않는
얼굴들이 제시되었을 때, 생성 AI가 이미 인간의 직관을 쉽게 속일 만큼
발전했다는 사실이 흥미로우면서도 놀라웠다.

특히 발표에서 지도 학습이 아닌 **비지도 학습**의 관점으로 생성 모델링을
설명한 부분이 인상적이었다. 생성 모델은 정답 없이도 데이터의 분포와
숨겨진 규칙을 학습하며, 단순한 분류를 넘어서 데이터 자체를 이해하려는
접근이라는 점이 의미 있었다.

잠재 변수(latent variable)를 플라톤의 동굴의 우화에 비유한 설명은 매우
직관적이었다. 우리가 보는 데이터는 그림자이고, 그 뒤에 있는 실제 구조가
잠재 변수라는 설명을 통해 생성 모델의 목표를 명확히 이해할 수 있었다.

오토인코더는 라벨 없이도 데이터를 압축하고 복원하는 과정에서 핵심 특징을
학습한다는 점에서 인상적이었다. 그러나 단순 복원에 그치는 한계가
존재하며, 이를 보완하기 위해 등장한 VAE는 잠재 변수를 확률 분포로
다룬다는 점에서 더 발전된 형태였다.

VAE에서 KL Divergence와 Reparameterization Trick을 통해 학습이
가능하도록 만든 부분은 생성 모델이 단순한 직관이 아닌 수학적으로 정교한
구조임을 보여주었다.

GAN은 생성자와 판별자가 경쟁하는 구조로, 매우 창의적인 접근이었다. 이
구조를 통해 점점 더 현실적인 데이터를 만들어낸다는 점에서 강한 인상을
받았다.

전체적으로 이번 발표는 생성 모델링의 흐름을 체계적으로 이해할 수 있도록
구성되어 있었으며, 생성 AI의 기초를 이해하는 데 매우 도움이 되었다.
앞으로 배우게 될 디퓨전 모델과 대형 언어 모델 또한 이러한 기반 위에서
발전한 것이라는 점에서, 이번 내용은 중요한 출발점이라고 생각한다.
