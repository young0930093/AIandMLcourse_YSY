# PRD — Physics Neural Network GUI App
**Version:** 1.0
**Date:** 2026-04-06
**Course:** AI와 머신러닝 — Week 4 과제

---

## 1. 목적 (Purpose)

TensorFlow/Keras로 작성된 4개의 물리 신경망 스크립트를 하나의 PySide6 GUI 애플리케이션으로 재구현한다.
사용자가 하이퍼파라미터를 조정하고 학습 과정을 실시간으로 시각화할 수 있는 인터랙티브 환경을 제공하여, 신경망이 물리 법칙을 학습하는 과정을 직관적으로 탐구할 수 있도록 한다.

---

## 2. 사용자 (Target User)

- AI/머신러닝 수업을 수강 중인 학부생
- 신경망 기초(레이어, 에폭, 손실함수)를 이해하고 있음
- Python 코드보다 GUI 환경에서 실험을 빠르게 반복하고 싶은 사용자

---

## 3. 핵심 기능 (Core Features)

### 3.1 탭 1 — 1D 함수 근사 (Perfect 1D)
- 3가지 함수(sin(x), cos(x)+0.5sin(2x), x·sin(x)) 중 선택하여 신경망으로 근사
- 네트워크 아키텍처(Small/Medium/Large/VeryLarge), 활성화 함수, Epochs, LR 조정
- 실시간 Loss curve + 학습 완료 후 함수 근사 그래프 및 오차 그래프 표시

### 3.2 탭 2 — 포물선 운동 회귀 (Projectile Motion)
- 입력(v₀, θ, t) → 출력(x, y) 관계를 신경망으로 학습
- 테스트 조건(v₀, θ) 슬라이더로 조정 후 예측 궤적 vs 실제 궤적 비교
- 각도별/속도별 오차 분석 그래프 제공

### 3.3 탭 3 — 과적합/과소적합 (Overfitting)
- 언더핏/굿핏/오버핏 3개 모델을 순차 학습
- 각 모델의 예측 결과와 Train/Val Loss curve 동시 비교
- Noise level 조정으로 과적합 심화 실험 가능

### 3.4 탭 4 — 진자 주기 예측 (Pendulum)
- 진자 길이(L)와 초기 각도(θ₀)로 주기(T) 예측 학습
- 학습 완료 후 RK4 수치 시뮬레이션으로 진자 운동 및 위상 공간 시각화
- NN 예측 주기 vs 물리 공식 주기 비교

## 4. 사용자 요구사항 (User Requirements)

### 필수 (Must Have)
| ID | 요구사항 |
|----|---------|
| UR-01 | Run 버튼을 누르면 선택한 하이퍼파라미터로 학습이 시작된다 |
| UR-02 | 학습 중 에폭마다 Loss 그래프가 실시간으로 업데이트된다 |
| UR-03 | Stop 버튼으로 학습을 중단할 수 있다 |
| UR-04 | 학습 완료 후 결과 그래프(예측 vs 실제, 오차)가 표시된다 |
| UR-05 | Epochs, Learning Rate, 네트워크 크기를 GUI에서 조정할 수 있다 |
| UR-06 | 학습 로그(에폭, 손실값)가 텍스트 영역에 출력된다 |
| UR-07 | 4개의 실험이 탭으로 구분된 하나의 앱으로 제공된다 |

### 선호 (Nice to Have)
| ID | 요구사항 |
|----|---------|
| UR-08 | Progress bar로 전체 학습 진행률을 표시한다 |
| UR-09 | 학습 중 다른 탭으로 전환해도 각 탭의 상태가 유지된다 |
| UR-10 | 오류 발생 시 사용자에게 명확한 메시지를 표시한다 |

- **호환성**: macOS, Windows에서 동작
- **의존성**: PySide6, TensorFlow, matplotlib, numpy (requirements.txt 제공)
