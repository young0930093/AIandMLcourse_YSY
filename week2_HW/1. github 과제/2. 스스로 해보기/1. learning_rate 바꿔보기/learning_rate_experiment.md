## 스스로 해보기 1: Learning Rate 실험

**실험 조건:** epochs=500, 용수철 데이터 (길이 = 2×무게 + 10)

| learning_rate | 예측식 | 15kg 예측 길이 | 이론값 | 결과 |
|---|---|---|---|---|
| 0.01 | 길이 = 2.02 × 무게 + 10.27 | 40.58 cm | 40.00 cm | 정상 학습 |
| 0.1 | nan | nan | 40.00 cm | 발산 |
| 1.0 | nan | nan | 40.00 cm | 발산 |

**분석:**

Learning rate는 경사하강법에서 한 번에 이동하는 보폭이다. 0.01일 때는 적절한 보폭으로 손실 함수의 최솟값에 안정적으로 수렴하여 실제 공식(2×무게+10)에 근접한 값(2.02×무게+10.27)을 학습했고, 15kg 예측도 이론값 40.00cm에 근접한 40.58cm가 나왔다.

반면 0.1과 1.0은 보폭이 너무 커서 최솟값을 지나쳐 반대편으로 튕기는 현상이 반복되었고, 결국 값이 무한대로 발산하여 nan(Not a Number)이 출력되었다.

이를 통해 learning rate가 너무 크면 학습이 불안정해지고, 적절히 작은 값을 선택하는 것이 중요함을 확인했다.
