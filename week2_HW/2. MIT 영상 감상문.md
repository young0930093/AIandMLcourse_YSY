# MIT 6.S191: RNN, 트랜스포머와 어텐션 강의 감상문

이번 강의는 순차적 데이터를 다루는 딥러닝 모델이 어떻게 발전해 왔는지를 RNN부터 트랜스포머까지 흐름 있게 설명해 주었다.

RNN이 과거 정보를 hidden state로 넘기면서 문맥을 기억한다는 아이디어 자체는 꽤 직관적이었다. 텍스트나 음성처럼 순서가 중요한 데이터에서 이전 입력이 다음 예측에 계속 영향을 준다는 구조가 자연스럽게 이해됐다.

하지만 시퀀스가 길어질수록 초기 정보가 희미해지고 기울기 소실 문제까지 생긴다는 한계는 실용적으로 꽤 치명적이라는 생각이 들었다. 순차 처리라 병렬화도 안 된다는 점도 속도 면에서 아쉬운 부분이었다.

가장 인상 깊었던 건 어텐션 메커니즘이다. Query, Key, Value 구조로 문장 안에서 단어 간 연관도를 직접 계산한다는 발상이 단순하면서도 강력했다. 이 덕분에 순차 처리 없이 병렬 연산이 가능해졌고, GPT 같은 LLM의 핵심 기반이 됐다는 게 새삼 와닿았다.

복잡한 수식보다 "왜 이 구조가 필요했는가"를 먼저 짚어준 강의 방식 덕에 전체 흐름을 자연스럽게 따라갈 수 있었다.
